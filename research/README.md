## Ausgangspunkte (aus initialem Brainstorm)

- Nanny/Childcare-Templates und Designs: TemplateMonster, ThemeForest, Dribbble, Webflow, Weblium, Themeslay, Vibeotter
- Trust-Signal-Guides: lineardesign.com, ivingocreative.com, site123.com
- Babysitting-Template-Vorlagen: Etsy, Freepik, MyCodelessWebsite

## Arbeitsweise

- **Jede Quelle mit Link** und kurzer Notiz (2–3 Zeilen) dokumentieren.
- **Keine blinde Übernahme** von Konkurrenz-Claims oder Designs.
- **Keyword-Recherche** idealerweise mit Tool (Google Keyword Planner, Ahrefs, Ubersuggest) — sonst manuell via Google Autosuggest, „People also ask", verwandte Suchanfragen.
